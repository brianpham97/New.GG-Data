## Available Scripts
Install required dependencies
```sh
npm install
```

Rename `example.env` in to `.env`\
Update `PORT` and `DB_NAME` in .env if needed

Start the server
```sh
npm run start
```

Load data into existing database
```sh
npm run load-data
```
