# Overview

Server and database to power [New.GG](https://github.com/brianpham97/New.GG-Front) application. Champion and skin data were extracted from Data Dragon.

## Available Scripts
Install required dependencies
```sh
npm install
```

Rename `example.env` to `.env`\
Update `PORT` and `DB_NAME` in .env if needed

Start the server
```sh
npm run start
```

Load data into existing database
```sh
npm run load-data
```
